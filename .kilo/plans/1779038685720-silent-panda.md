# Plan: Faculty SHS — Regular/Admin Hours + Admin Pay from Authoritative Sources

## Problem

`api/attendance.php` (the unified router, the one the UI actually calls) has two bugs in both `handleFacultySHS()` and `handleFacultyCollege()`:

1. **GET** returns stale values from `attendance_faculty_shs/faculty_college` with no sync from the source tables.
2. **POST** computes `gross_pay` using hardcoded rates (70 for SHS, 85 for College) multiplied against the posted admin hours, rather than reading the already-stored `attendance_admin_pay.total_pay`.

The standalone `api/attendance/faculty-shs.php` already handles this correctly by reading `regular_hours` from `attendance_shs_dtr.total_hours` and `admin_hours`/`total_pay` from `attendance_admin_pay`. The unified router (`api/attendance.php`) must do the same.

---

## Source-of-Truth Table

| Faculty SHS Column | Reads From | Field |
|---|---|---|
| Regular Hrs | SHS DTR | `attendance_shs_dtr.total_hours` |
| Admin Hrs | Admin Pay | `attendance_admin_pay.admin_hours` |
| Admin Pay | Admin Pay | `attendance_admin_pay.total_pay` ← already stored by Admin Pay tab |
| Gross Pay | computed | `regular_hours × 80 + admin_pay.total_pay` |

| Faculty College Column | Reads From | Field |
|---|---|---|
| Regular Hrs | College DTR | `attendance_college_dtr.total_hours` |
| Admin Hrs | Admin Pay | `attendance_admin_pay.admin_hours` |
| Admin Pay | Admin Pay | `attendance_admin_pay.total_pay` ← already stored by Admin Pay tab |
| Gross Pay | computed | `regular_hours × 85 + admin_pay.total_pay` |

## Files to Modify

### File 1 — `api/attendance.php`

#### Addition: shared private sync helpers

Add two private methods to `AttendanceRouter` that mirror the standalone `fetchSyncedHoursAndAdminPay()`:

```php
// SHS source: attendance_shs_dtr for regular_hours
private function fetchSyncedSHS(string $empId, string $start, string $end): array {
    $regular = 0; $adminHours = 0; $adminTotalPay = 0;

    // Regular hours from SHS DTR
    $stmt = $this->pdo->prepare("
        SELECT COALESCE(total_hours, 0) AS regular_hours
        FROM attendance_shs_dtr
        WHERE employee_id = :id AND period_start = :start AND period_end = :end LIMIT 1
    ");
    $stmt->execute([':id' => $empId, ':start' => $start, ':end' => $end]);
    $regular = (float)($stmt->fetch()['regular_hours'] ?? 0);

    // Admin hours + total_pay from Admin Pay (authoritative admin pay stored by Admin Pay tab)
    $stmt = $this->pdo->prepare("
        SELECT COALESCE(admin_hours, 0) AS admin_hours, COALESCE(total_pay, 0) AS total_pay
        FROM attendance_admin_pay
        WHERE employee_id = :id AND period_start = :start AND period_end = :end LIMIT 1
    ");
    $stmt->execute([':id' => $empId, ':start' => $start, ':end' => $end]);
    $row = $stmt->fetch();
    $adminHours    = (float)($row['admin_hours'] ?? 0);
    $adminTotalPay = (float)($row['total_pay'] ?? 0);

    return ['regular_hours' => $regular, 'admin_hours' => $adminHours, 'admin_total_pay' => $adminTotalPay];
}

// College source: attendance_college_dtr for regular_hours
private function fetchSyncedCollege(string $empId, string $start, string $end): array {
    $regular = 0; $adminHours = 0; $adminTotalPay = 0;

    $stmt = $this->pdo->prepare("
        SELECT COALESCE(total_hours, 0) AS regular_hours
        FROM attendance_college_dtr
        WHERE employee_id = :id AND period_start = :start AND period_end = :end LIMIT 1
    ");
    $stmt->execute([':id' => $empId, ':start' => $start, ':end' => $end]);
    $regular = (float)($stmt->fetch()['regular_hours'] ?? 0);

    $stmt = $this->pdo->prepare("
        SELECT COALESCE(admin_hours, 0) AS admin_hours, COALESCE(total_pay, 0) AS total_pay
        FROM attendance_admin_pay
        WHERE employee_id = :id AND period_start = :start AND period_end = :end LIMIT 1
    ");
    $stmt->execute([':id' => $empId, ':start' => $start, ':end' => $end]);
    $row = $stmt->fetch();
    $adminHours    = (float)($row['admin_hours'] ?? 0);
    $adminTotalPay = (float)($row['total_pay'] ?? 0);

    return ['regular_hours' => $regular, 'admin_hours' => $adminHours, 'admin_total_pay' => $adminTotalPay];
}
```

## Step-by-step

1. **Add two private sync helpers** (`fetchSyncedSHS`, `fetchSyncedCollege`) to `AttendanceRouter` in `api/attendance.php`.
2. **`handleFacultySHS()` GET:** after `$stmt->fetchAll()` loop, overwrite each row's `regular_hours`, `admin_hours`, `admin_total_pay` using `fetchSyncedSHS()`. Recompute `gross_pay` and `net_pay` before returning.
3. **`handleFacultySHS()` POST:** discard posted `regular_hours`/`admin_hours`; call `fetchSyncedSHS()`; compute `gross = syncedRegular * 80 + syncedAdminTotalPay`; save synced hours + admin pay as authoritative values.
4. **`handleFacultyCollege()` GET/POST:** same pattern using `fetchSyncedCollege()`, with regular rate 85.
5. **`attendance.html` `TAB_CONFIG`:** set `editable[0]` to `false` (`regular_hrs`) for both faculty tabs.

## Resulting behaviour

```
Faculty SHS displays:
  Regular Hrs  ← attendance_shs_dtr.total_hours   (edited in SHS-DTR tab)
  Admin Hrs    ← attendance_admin_pay.admin_hours (edited in Admin Pay tab)
  Admin Pay    ← attendance_admin_pay.total_pay   (stored by Admin Pay tab, reflects whatever admin rate the user entered in the Admin Pay tab, *not* a hardcoded 70)

Gross Pay = Regular Hrs × 80  +  attendance_admin_pay.total_pay
```

```
Faculty College displays:
  Regular Hrs  ← attendance_college_dtr.total_hours (edited in College-DTR tab)
  Admin Hrs    ← attendance_admin_pay.admin_hours   (edited in Admin Pay tab)
  Admin Pay    ← attendance_admin_pay.total_pay      (stored by Admin Pay tab)

Gross Pay = Regular Hrs × 85  +  attendance_admin_pay.total_pay
```
